# Notion データベース作成ガイド

このガイドに従って、Notion に 3 つのデータベースを作成し、ダミーデータを入力してください。

## 📋 事前準備

### Step 1: Notion アカウントを作成（未作成の場合）

1. [Notion 公式サイト](https://www.notion.so) にアクセス
2. 「Sign up」をクリック
3. メールアドレスで登録

### Step 2: Notion ワークスペースを作成

1. ログイン後、「Create workspace」をクリック
2. ワークスペース名を入力
3. 「Create」をクリック

---

## 🗄️ データベース 1: スタートアップDB

### Step 1: データベースを作成

1. Notion ワークスペースを開く
2. 左サイドバーの「+ Add a page」をクリック
3. 「Database」を選択
4. 「Table」を選択
5. 名前を入力: `スタートアップDB`

### Step 2: プロパティを設定

以下のプロパティを追加します：

| プロパティ名 | タイプ | 説明 |
|------------|--------|------|
| 企業名 | Title | スタートアップの名前（デフォルト） |
| 出身スタジオ | Select | 出身スタジオ名 |
| 領域 | Select | HR Tech, Fin Tech など |
| 設立年 | Number | 設立年 |
| 調達額 | Number | 累計調達額（億円） |
| ステージ | Select | Seed, Series A など |
| 説明 | Text | 企業の説明文 |
| Exit | Select | IPO, M&A など |

**プロパティ追加方法:**

1. テーブルの右側の「+」をクリック
2. プロパティ名を入力
3. タイプを選択
4. 「Done」をクリック

## 📄 PDF ファイルについて

PDF ファイルがない場合、以下の方法で対応できます：

### 方法 1: サンプル PDF を作成

1. テキストエディタで簡単なテキストを作成
2. 「Print to PDF」で PDF に変換
3. Notion にアップロード

### 方法 2: PDF ファイルをスキップ

- 「PDFファイル」プロパティを空のままにしておく
- 後で追加することもできます

## 🔗 データベース URL を取得

各データベースの共有リンクを取得してください：

### Step 1: データベースを開く

1. Notion ワークスペースから該当のデータベースを開く

### Step 2: 共有ボタンをクリック

1. 右上の「Share」をクリック
2. 「Copy link」をクリック

### Step 3: リンクを保存

以下の形式で 3 つのリンクを保存：

```
スタートアップDB: https://notion.so/...
スタジオ一覧: https://notion.so/...
レポート: https://notion.so/...
```

## 🔑 Notion Integration Token を取得

### Step 1: Notion Developers にアクセス

1. [Notion Developers](https://www.notion.so/my-integrations) にアクセス
2. 「New integration」をクリック

### Step 2: Integration を作成

1. Name: `スタートアップスタジオ協会`
2. Associated workspace: 使用するワークスペースを選択
3. 「Submit」をクリック

### Step 3: Token をコピー

1. 表示される「Internal Integration Token」をコピー
2. 安全に保管（後で Vercel に設定）

### Step 4: データベースへのアクセス権限を付与

各データベースで以下を実行：

1. データベースを開く
2. 右上の「…」をクリック
3. 「Add connections」を選択
4. 作成した Integration を選択

**Notion データベース作成完了後、以下の情報を Vercel に設定してください：**

```
NOTION_API_KEY=your_token
NOTION_STARTUPS_DB_ID=database_id
NOTION_STUDIOS_DB_ID=database_id
NOTION_REPORTS_DB_ID=database_id
```

詳細は VERCEL_DEPLOY.md を参照してください。
